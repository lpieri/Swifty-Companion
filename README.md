## Description

Swifty Companion is not available on the AppStore because it's a project to learn Swift and Xcode.

This application contain:
 - 42 Api
 - 2 View
 - Search Bar
 - ...

## Technologies

The technologies I used to make this project happen are:
 - SwiftUI
 - Swift 5
 - Xcode 11.4
 - iOS 13.4
 - OAuth2 Framework

## Bonus

 - Dark Mode
 - Design View
 - Custom Cell
 - Custom Scroll View
 - Custom Pop Up
